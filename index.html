🍺 Editors and Beers

A conference travel planning tool. Enter a venue or city, set a radius, and see nearby breweries, coffee shops, and researchers on an interactive map.

Data sources

Brewery locations come from Open Brewery DB (openbrewerydb.org) for US/Canada results, and OpenStreetMap via Overpass API for international results. Coffee shops use OpenStreetMap data. Coverage quality varies by region.

Editors layer

Upload a CSV with columns: Editor, Journal, Role, Country, Affiliation — optional: City, Email, Website. Locations are resolved from a built-in institution database, Nominatim geocoding, or country-level fallback.

Privacy

No data is stored. No analytics. Refreshing the page clears all state.
